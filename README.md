# MediSlot

MediSlot is a modern medical appointment booking system built using FastAPI, HTML, CSS, and JavaScript.

## Features

- JWT Authentication
- Patient Registration/Login
- Doctor Slider
- Appointment Booking
- Slot Conflict Validation
- My Appointments
- Notifications
- Doctor Search
- Responsive UI

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- FastAPI
- SQLAlchemy
- SQLite

Authentication:
- JWT
- Passlib

## Run Backend

uvicorn app:app --reload

## Frontend

Run using Live Server.
